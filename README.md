# fluttergo

# 项目介绍
项目官网sharedgrow.com.本项目使用dart的flutter框架作为前端页面，golang的goframe框架作为后端服务，使用mysql数据库和redis。
作为一个融合了flutter和golang的项目，该项目实现了web页面展示功能,用户可根据需要自行修改成支持多终端的跨平台应用.


# 框架版本
flutter: 3.27.1
goframe: 2.9.0
mysql: 8.0.12
redis: 3.0.504

# 项目功能
前端正在改进中.目前实现了.用户注册,登录,用户资料修改,权限管理,文章发布和管理,菜单管理等功能。
